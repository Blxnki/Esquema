En javaSCript lo puedes escribir dentro de la estructura del html, pero las buenas practicas de programación dicen que es mejor y mas eficiente crear un archivo separado para el código javascript, ya que el navegador tiene que cargar todo el archivo html antes de poder ejecutar el código javascript, por lo que si tienes mucho código javascript, el navegador tendrá que esperar mucho tiempo para poder cargar la página.
Para crear un archivo tiene que que ser .js por ejemplo: script.js Despues de crear el archivo tienes que llamarlo en el archivo html con la etiqueta <script src="script.js"></script> Es recomendable ponerlo al final en la estructura del archivo html para que el navegador pueda cargar la página antes de ejecutar el código javascript.
En esta guia puedes ver que hay un arhivo .js que es el archivo que te da la bienvenida al principio.
Una gran recomendasion para programar es que uses un editor de texto como sublime text o atom o visual studio code ya que estos editores te permiten crear y editar archivos de manera mucho mas eficiente y rapida que el bloc de notas de windows o el bloc de notas de mac.
