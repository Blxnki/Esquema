Objetos

JavaScript es un lenguaje de programación orientado a objetos que permite modelar entidades del mundo real como objetos. Esta característica facilita el desarrollo y mantenimiento de software, ya que se trabaja con conceptos similares a los del mundo físico.
Los objetos en JavaScript se componen de:
Propiedades: Son los rasgos distintivos o cualidades que describen al objeto. Por ejemplo, para un coche, las propiedades podrían ser marca, modelo, color y matrícula. Cada propiedad tiene un nombre y un valor.
Métodos: Determinan el comportamiento del objeto, es decir, las acciones que puede realizar. Para un coche, los métodos podrían ser arrancar, frenar y climatizar. Los métodos son funciones asociadas al objeto.
Los objetos son tipos de datos compuestos, formados por una colección de valores (propiedades), mientras que los datos primitivos solo contienen un valor.
Los objetos son mutables, pudiendo modificarse sin cambiar su identidad, mientras que los valores primitivos son inmutables.
En JavaScript, se trabaja con referencias a objetos, lo que tiene implicaciones importantes en la asignación, comparación y paso de objetos como argumentos a funciones.
Para crear un objeto en JavaScript, se puede utilizar la siguiente sintaxis:
var nombreObjeto = {
                    propiedad1: valor1,
                    propiedad2: valor2,
                    metodo1: function() {
                      // Código del método
                    }
                  };

Propiedades

Los objetos en JavaScript son estructuras de datos que permiten almacenar colecciones de propiedades (también llamadas atributos) y métodos. Las propiedades son características o cualidades del objeto, mientras que los métodos son funciones asociadas al objeto.
Para acceder a las propiedades de un objeto, se pueden utilizar dos notaciones:
Notación de punto: objeto.propiedad
Notación de corchete: objeto['propiedad']
Para modificar las propiedades de un objeto, se pueden utilizar las mismas notaciones visto anteriormente.
objeto.propiedad = "lo que quieres cambiar"
Para eliminar una propiedad, se utiliza el operador delete:
delete objeto.propiedad
Para recorrer todas las propiedades de un objeto, se puede usar el bucle for...in:
for (let propiedad in objeto) { console.log(propiedad + ": " + objeto[propiedad]); }

Métodos

Los métodos en la programación orientada a objetos (POO) representan las acciones que pueden realizar los objetos de una clase. Estos métodos definen el comportamiento de los objetos y constituyen su interfaz, permitiendo la comunicación con otros objetos o clases.
Los métodos se definen dentro de una clase y se implementan como funciones. En la notación de objetos, los métodos se representan como pares clave:valor, donde la clave es el nombre del método y el valor es una función anónima que contiene las instrucciones que establecen su comportamiento.
Características principales
Acceso a atributos: Los métodos pueden acceder y modificar los atributos del objeto utilizando la palabra clave this o self, dependiendo del lenguaje de programación.
ipos de métodos: Existen varios tipos de métodos, incluyendo:
Métodos para recuperar el estado del objeto
Métodos para modificar el estado del objeto
Métodos para realizar acciones específicas
Constructores para crear instancias de objetos
Invocación: Los métodos se invocan utilizando la sintaxis objeto.metodo(argumentos).

LA FUNCIÓN CONSTRUCTORA

Las funciones constructoras en JavaScript son una forma eficiente de crear objetos dinámicamente, especialmente cuando se necesitan múltiples instancias de un mismo tipo de objeto. Estas funciones ofrecen ventajas significativas sobre la notación literal de objetos en escenarios donde los datos no se conocen de antemano o cuando se requiere crear objetos en tiempo de ejecución.
Se invocan con el operador new.
Convencionalmente, su nombre comienza con mayúscula.
Crean un objeto vacío y lo asignan a this antes de ejecutar el cuerpo de la función.
Permiten definir propiedades y métodos comunes para todos los objetos creados con ellas.
Ventajas de las funciones constructoras
Creación dinámica de objetos: Permiten crear objetos cuando los datos se obtienen en tiempo de ejecución, por ejemplo, a través de formularios.
Eficiencia en la creación de múltiples objetos: Facilitan la creación de numerosos objetos del mismo tipo con una sola línea de código por objeto.
Reutilización de código: La estructura del objeto se define una vez en la función constructora y se reutiliza para cada nueva instancia.
Flexibilidad: Permiten añadir propiedades y métodos específicos a instancias individuales después de su creación.
El operador instanceof se utiliza para verificar si un objeto fue creado por una función constructora específica

MUTABILIDAD

La mutabilidad en JavaScript se refiere a la capacidad de modificar un objeto después de su creación. A diferencia de los tipos de datos primitivos (como números, strings y booleanos) que son inmutables, los objetos (incluyendo arrays) son mutables, lo que significa que su estado interno puede cambiar.
Mutable: Un objeto mutable puede ser modificado después de su creación.
Inmutable: Un objeto inmutable no puede ser modificado después de su creación.
Los datos primitivos en JavaScript son inmutables.
Los objetos y arrays son mutables.
Para evitar la mutación directa de objetos, se pueden clonar. JavaScript ofrece varias formas de clonar objetos:
Operador Spread: Crea una copia superficial del objeto.
const copy1 = { ...data };
Object.assign(): Crea una copia superficial del objeto.
const copy2 = Object.assign({}, data);
structuredClone(): Crea un clon profundo de un valor dado utilizando el algoritmo de clonación estructurada.
const copy4 = structuredClone(data);
Copia Superficial: Copia el primer nivel del objeto, pero los niveles más profundos se referencian.
Copia Profunda: Crea una copia real para los objetos anidados.
JavaScript ofrece varias maneras de asegurar la inmutabilidad de los objetos:
Object.assign(): Permite crear un nuevo objeto copiando los valores de otro objeto.
Object.freeze(): "Congela" un objeto, impidiendo que sea cambiado.
Sintaxis Spread: Crea un nuevo objeto con los valores de otro objeto.

Objetos Globales

JavaScript permite representar tipos de datos primitivos como objetos, aunque en la práctica se recomienda trabajar directamente con los valores primitivos por razones de rendimiento y simplicidad.
Los tipos primitivos (number, string, boolean) pueden ser representados como objetos usando sus respectivos constructores (Number, String, Boolean).
La comparación entre objetos primitivos y valores primitivos puede dar resultados inesperados:
let numero1 = 5; let numero2 = new Number(5); console.log(numero1 == numero2); // true console.log(numero1 === numero2); // false
Dos objetos primitivos con el mismo valor nunca serán iguales al compararlos:
let obj1 = new Number(5); let obj2 = new Number(5); console.log(obj1 == obj2); // false console.log(obj1 === obj2); // false
Los objetos primitivos ofrecen propiedades y métodos útiles:
Number: MAX_VALUE, MIN_VALUE, MAX_SAFE_INTEGER, MIN_SAFE_INTEGER
Métodos de conversión: toString(), parseInt(), parseFloat()
JavaScript proporciona varios objetos globales útiles:
Array: Para almacenar conjuntos ordenados de elementos
Date: Para representar y manipular fechas y horas
Math: Para realizar operaciones matemáticas
String: Para procesamiento avanzado de texto

El Objeto Array

Los arrays en JavaScript son estructuras de datos que permiten almacenar una colección ordenada de elementos.
Usando corchetes (notación literal):
const bebidas = ["agua", "cerveza", "vino"];
Usando el constructor Array():
const bebidas = new Array("agua", "cerveza", "vino");
Como un objeto con índices numéricos:
const bebidas = { 0: "agua", 1: "cerveza", 2: "vino" };
Características importantes de los arrays en JavaScript:
Pueden contener elementos de cualquier tipo, incluso otros objetos o arrays.
Se recomienda declararlos como constantes (const) para evitar reasignaciones accidentales, aunque esto no impide modificar sus elementos internos.
El operador typeof devuelve "object" para los arrays, por lo que para verificar si una variable es un array, se usa el operador instanceof
Los arrays en JavaScript son muy flexibles y ofrecen numerosos métodos para manipular y transformar datos, como forEach(), map(), filter(), y reduce(), entre otros.

Acceso a los elementos de un array

Para acceder y manipular los elementos de un array en JavaScript, se utilizan los siguientes métodos:
Acceso a elementos:
Se usa el índice entre corchetes: array[posición], el índice comienza en 0
Modificar o agregar elementos:
Se usa la misma notación: array[posición] = nuevoValor
ejemplo: bebidas[3] = "Coca-Cola" agrega un nuevo elemento
Eliminar elementos:
Se usa el operador delete: delete array[posición]
Longitud del array:
Se usa la propiedad length: array.length
Recorrer elementos:
Usando FOR:
for (let i = 0; i = bebidas.length; i++) { console.log(bebidas[i]); }
Usando for..of:
for (let elemento of bebidas) { console.log(elemento); }
Usando foreach():
bebidas.forEach(function(elemento, indice, array) { console.log(elemento); });
Convertir array a string:
Método toString(): bebidas.toString()

Métodos de unión, división y transformación de arrays

Los métodos concat(), slice(), y map() en JavaScript permiten crear nuevos arrays a partir de arrays existentes sin modificar los originales.
concat(): Crea un nuevo array combinando elementos de otros arrays o valores:
const bebidasSinAlcohol = ["agua", "zumo"]; const bebidasConAlcohol = ["cerveza", "vino"]; const bebidas = bebidasSinAlcohol.concat(bebidasConAlcohol); console.log(bebidas); // ["agua", "zumo", "cerveza", "vino"]
slice(): Extrae una porción de un array existente para crear uno nuevo:
const bebidas = ["agua", "zumo", "cerveza", "vino"]; const bebidasSinAlcohol = bebidas.slice(0, 2); const bebidasConAlcohol = bebidas.slice(2); console.log(bebidasSinAlcohol); // ["agua", "zumo"] console.log(bebidasConAlcohol); // ["cerveza", "vino"]
map(): Crea un nuevo array aplicando una función a cada elemento del array original:
const temperaturasCentigrados = [0, 15, 30, 45]; const temperaturasFahrenheit = temperaturasCentigrados.map(function(temp) { return temp * 9/5 + 32; }); console.log(temperaturasFahrenheit); // [32, 59, 86, 113]

Métodos de ordenación de los elementos de un array

JavaScript proporciona varios métodos para ordenar y buscar elementos en un array.
sort(): Ordena un array alfabéticamente según los valores Unicode de los caracteres.
every(función): Retorna true si todos los elementos cumplen una condición.
some(función): Retorna true si algún elemento cumple la condición.
filter(función): Devuelve un nuevo array con los elementos que cumplen la condición.
find(función): Devuelve el primer elemento que cumple la condición.
findIndex(función): Retorna la posición del primer elemento que cumple la condición.
indexOf(elemento): Retorna la primera posición de un elemento en el array (-1 si no existe).
lastIndexOf(elemento): Retorna la última posición de un elemento en el array.
includes(elemento): Retorna true si el array contiene el elemento, incluso si es NaN.

Acceso a los caracteres de una cadena

Métodos de manejo de cadenas

concat(): Une dos o más cadenas.
let saludo = "Hola"; let nombre = "Mundo"; console.log(saludo.concat(", ", nombre, "!")); // "Hola, Mundo!"
slice(inicio, fin): Extrae una parte de la cadena. Admite índices negativos.
let texto = "JavaScript"; console.log(texto.slice(0, 4)); console.log(texto.slice(-6));
substring(inicio, fin): Similar a slice(), pero no admite índices negativos.
console.log(texto.substring(0, 4));
replace(valor, nuevoValor): Reemplaza la primera coincidencia.
let frase = "Me gusta JavaScript"; console.log(frase.replace("JavaScript", "Python"));
toUpperCase(): Convierte a mayúsculas.
toLowerCase(): Convierte a minúsculas.
let palabra = "Hola"; console.log(palabra.toUpperCase()); console.log(palabra.toLowerCase());
trim(): Elimina espacios en blanco al inicio y al final.
trimStart(): Elimina solo al inicio.
trimEnd(): Elimina solo al final.
let mensaje = " Hola Mundo "; console.log(mensaje.trim());
split(separador): Divide una cadena en un array.
let lista = "manzana,pera,plátano"; console.log(lista.split(","));
indexOf(valor): Devuelve la primera posición de valor o -1 si no lo encuentra.
lastIndexOf(valor): Busca desde el final.
search(expresiónRegular): Similar a indexOf(), pero acepta expresiones regulares.
includes(valor): Devuelve true si la cadena contiene valor.
startsWith(valor): Verifica si la cadena comienza con valor.
endsWith(valor): Verifica si la cadena termina con valor.
let frase2 = "JavaScript es genial"; console.log(frase2.includes("JavaScript")); console.log(frase2.startsWith("Java")); console.log(frase2.endsWith("genial"));

Unión, división y sustitución de partes de una cadena

Para reemplazar texto, replace() solo cambia la primera coincidencia, a menos que usemos expresiones regulares (/g).

El objeto Date

El objeto Date en JavaScript es fundamental para manejar fechas y horas. Se usa para registrar eventos, calcular diferencias de tiempo, validar fechas y más.
El constructor new Date() permite crear objetos de fecha y hora de varias maneras:
let ahora = new Date(); console.log(ahora);
Puedes poner una fecha específica:
let fechaISO = new Date("2023-08-11"); console.log(fechaISO);
Especificando fecha y hora
let fechaHora = new Date("2023-08-11T12:56:00Z"); // Z indica UTC console.log(fechaHora);
Crear una fecha con valores numéricos
let fechaNumerica = new Date(2023, 7, 11, 12, 56, 0); console.log(fechaNumerica);
Fechas basadas en milisegundos desde 1970
let fechaMilis = new Date(0); // 1 de enero de 1970 console.log(fechaMilis);
Métodos para Obtener Información de una Fecha
let fecha = new Date("2025-03-23T15:30:45Z"); console.log(fecha.getFullYear()); // 2025 console.log(fecha.getMonth()); // 2 (Marzo, recuerda que enero = 0) console.log(fecha.getDate()); // 23 console.log(fecha.getDay()); // 0 (Domingo, cuenta desde 0 = Domingo) console.log(fecha.getHours()); // 15 console.log(fecha.getMinutes()); // 30 console.log(fecha.getSeconds()); // 45 console.log(fecha.getMilliseconds()); // 0 console.log(fecha.getTime()); // Milisegundos desde 1970

Métodos de manejo de fechas

El objeto Date en JavaScript proporciona varios métodos para obtener y establecer la fecha y la hora, tanto en formato local como en formato UTC. A continuación, se detallan los métodos más comunes que permiten trabajar con fechas y horas.
getFullYear(): Devuelve el año de la fecha (4 dígitos).
let fecha = new Date(); console.log(fecha.getFullYear()); // Ej: 2025
getMonth(): Devuelve el mes (rango 0-11). Enero es 0, diciembre es 11.
console.log(fecha.getMonth()); // Ej: 2 para marzo
getDate(): Devuelve el día del mes (rango 1-31).
console.log(fecha.getDate()); // Ej: 23
getDay(): Devuelve el día de la semana (rango 0-6). Domingo es 0, sábado es 6.
console.log(fecha.getDay()); // Ej: 6 para sábado
getHours(): Devuelve la hora (rango 0-23).
console.log(fecha.getHours()); // Ej: 15
getMinutes(): Devuelve los minutos (rango 0-59).
console.log(fecha.getMinutes()); // Ej: 30
getSeconds(): Devuelve los segundos (rango 0-59).
console.log(fecha.getSeconds()); // Ej: 45
getTime(): Devuelve el número de milisegundos desde el 1 de enero de 1970.
console.log(fecha.getTime()); // Ej: 1648033200000
setFullYear(año, mes, día): Establece el año, mes (opcional) y día (opcional).
fecha.setFullYear(2025); // Cambia el año a 2025
setMonth(mes, día): Establece el mes (rango 0-11) y opcionalmente el día.
fecha.setMonth(2); // Cambia el mes a marzo (mes 2)
setDate(día): Establece el día del mes (rango 1-31).
fecha.setDate(15); // Cambia el día a 15
setHours(hora, minuto, segundo, milisegundo): Establece la hora (rango 0-23), minutos, segundos y milisegundos.
fecha.setHours(18, 30, 0, 0);
setMinutes(minuto, segundo, milisegundo): Establece los minutos (rango 0-59), segundos y milisegundos.
fecha.setMinutes(45); // Cambia los minutos a 45
setSeconds(segundo, milisegundo): Establece los segundos (rango 0-59) y los milisegundos.
fecha.setSeconds(30); // Cambia los segundos a 30
setTime(milisegundos): Establece la fecha basada en el número de milisegundos desde el 1 de enero de 1970.
fecha.setTime(1648033200000);
