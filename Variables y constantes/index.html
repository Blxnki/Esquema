Unidad 3

Variables y constantes

Una variable es un espacio de memoria para almacenar información.
Se declaran con la palabra clave let seguida del identificador.
Ejemplo: let identificador
Debes seguir ciertas reglas para declarar variables: Pueden contener letras, dígitos, $ y _.
No pueden empezar con un dígito.
No se pueden usar palabras reservadas.
Son sensibles a mayúsculas y minúsculas.
Se recomienda usar camelCase para nombres compuestos.Ejemplo Let nombreDeLaVariable
Cuando declaras una variable es recomendable inicializarla con un valor, ya que esto te permite saber que valor tiene la variable. La varibles sin inicializar tienen el valor: undefined.
Como dije en la unidad anterior las buenas practicas de la programación es importante usar "use strict". Tambien declarar siempre las variables antes de usarlas.
JavaScript es de tipado dinámico: una variable puede cambiar de tipo. Como dije anteriormente, si en el navegador web pulsas F12 y vas a la pestaña "console" y veras el resultado y los errores del codigo.
Una constante es un espacio de memoria para almacenar datos cuyo valor no puede cambiar después de ser asignado. Se declaran con la palabra clave const, seguida de un identificador y un valor inicial obligatorio. Ejemplo: const PI = 3.14;
No pueden ser reasignadas ni redeclaradas dentro del mismo ámbito. Es obligatorio inicializarlas al declararlas. Ejemplo incorrecto: const x; // Error: falta inicialización.
Se recomienda escribir los nombres de constantes en mayúsculas y separar palabras con guiones bajos (_). Ejemplo: const MAX_VALUE = 100;
