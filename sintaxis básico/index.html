sintaxis básico

Las sentencias se separan entre sí con un punto y coma (ʻ;ʼ). 
                    
sentencias1;

                    
sentencias2;
Las sentencias en JavaScript están compuestas por los siguientes elementos:
Valores: Datos que manejan los programas, como booleanos (true/false), números y textos. Estos son tipos
primitivos porque representan un único dato.
Variables o constantes: Almacenan valores que pueden ser referenciados posteriormente.
Operadores: Permiten combinar valores y variables para formar expresiones, como 2 + 3 (resultado: 5).
Palabras clave: Indican acciones específicas, como let, que se usa para declarar variables.
Todo lo escrito en un script puede ejecutarse, excepto si es un comentario. Los comentarios sirven para explicar el código y facilitar su mantenimiento o depuración.
Comentarios de una línea: Usan //.
Comentarios de varias líneas: Empiezan con /* y terminan con */.
NOTA IMPORTANTE!!:Usa "use strict" esto nos ayudara a escribir código más seguro y con menos errores, ya que esto impone un conjunto más estricto de reglas para escribir JavaScript, generando más advertencias y lanzando más errores.
Para ver los ejemplos accede al archivo .js o desde la pagina web pulsando F12 y yendo a la terminal.
