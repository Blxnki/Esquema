Tipos de datos primitivos

JavaScript utiliza tipado dinámico, lo que significa que no es necesario declarar el tipo de una variable al crearla. Los tipos primitivos son los más básicos con los que se puede trabajar, mientras que los tipos compuestos están formados por uno o más datos primitivos.
Estos tipos primitivos son inmutables, lo que significa que sus valores no pueden ser modificados una vez creados. Sin embargo, las variables que contienen estos valores pueden ser reasignadas.
Los tipos de datos primitivos en JavaScript son:
Boolean: Representa valores lógicos true o false.
Number: Abarca números enteros y decimales.
BigInt: Para números enteros más grandes que los que puede representar Number.
String: Cadenas de caracteres, pueden usar comillas dobles o simples indistintamente.
Null: Representa un valor nulo o vacío.
Undefined: Indica la ausencia de valor, asignado automáticamente a variables declaradas sin inicializar.
Symbol: Un tipo adicional no cubierto en detalle en este contexto.

Boolean

El tipo de datos Boolean en JavaScript representa los conceptos de verdadero (true) y falso (false). Estas son sus características principales:
Valores: Solo tiene dos posibles valores: true y false.
Uso en comparaciones: Son el resultado de evaluar expresiones de comparación y operaciones lógicas.
Se utilizan en estructuras de control como:
Sentencias condicionales: if...else y switch...case.for, while, y do...while.

Number

JavaScript usa un solo tipo Number para todos los valores numéricos (enteros y decimales)./div>
Limitaciones de precisión:
Fuera de ese rango, se producen redondeos.
Los cálculos con decimales pueden tener imprecisiones (ej: 0.1 + 0.2 no da exactamente 0.3).
Para solucionar esto se pone una n al final commo por ejemplo: 9007199254740993n
La "n" es para usar el tipo BigInt, lo malo es que no maneja decimales
Valores especiales:
Infinity y -Infinity para representar infinito matemático.
NaN (Not a Number) para operaciones matemáticas inválidas.

String

El tipo de datos String se utiliza para manejar textos, representados como una secuencia de caracteres alfanuméricos entre comillas simples (') o dobles (").
Ejemplo: 'Hola' y "Hola" son equivalentes.
Se pueden mezclar comillas simples y dobles dentro de una cadena, siempre que sean diferentes.
Ejemplo: 'El libro "Don Quijote"'
Permiten incluir caracteres especiales dentro de las cadenas utilizando una barra invertida (\).
Ejemplos comunes:
\n: Salto de línea.
\t: Tabulador.
\': Comilla simple.
\\: Barra invertida literal.
Las cadenas son inmutables, lo que significa que no pueden modificarse directamente, pero es posible crear nuevas cadenas basadas en operaciones sobre las existentes.
Para dividir cadenas largas en varias líneas en el código, se debe usar el carácter \ al final de cada línea.

NULL Y UNDEFINED

NULL y UNDEFINED son dos valores especiales en JavaScript que se utilizan para representar la ausencia de un valor.
NULL es un valor que se utiliza para representar la ausencia de un valor en un campo de una base de datos o en un objeto.
UNDEFINED es un valor que se utiliza para representar la ausencia de un valor en una variable o en un objeto.

Conversión a números

Number(valor): Convierte cualquier tipo de dato primitivo a número.
Los espacios al inicio o final de una cadena no afectan la conversión.
Si el valor no puede ser convertido a número, devuelve NaN.
Cadenas vacías o solo con espacios se convierten a 0.
La conversión es automática en operaciones aritméticas, excepto en la suma.
parseInt(cadena, base): Convierte una cadena a número entero.
parseFloat(cadena, base): Convierte una cadena a número de coma flotante.
Admiten un segundo argumento opcional para especificar la base numérica.
parseInt solo devuelve NaN si la cadena no comienza por un número.
parseInt elimina decimales sin redondear.

Conversión a cadenas

La función global String(valor) es la principal forma de convertir cualquier tipo de dato primitivo en una cadena en JavaScript.
Números: String(123) devuelve "123".
Booleanos: String(true) devuelve "true", String(false) devuelve "false"
Valores especiales: String(null) devuelve "null", String(undefined) devuelve "undefined"
Alternativamente, para booleanos y números, se puede usar el método toString():
123.toString() devuelve "123".
true.toString() devuelve "true".
Existen otros métodos específicos para números que ofrecen más control sobre la conversión:
toFixed(decimales): Redondea el número a los decimales especificados.
toExponential(decimales): Expresa el número en notación exponencial.
toPrecision(dígitos): Redondea el número al número de dígitos significativos especificado.

Conversión a booleanos

Boolean(valor): Convierte cualquier tipo de dato primitivo a booleano.
Cadenas:
Cualquier cadena no vacía (incluso si solo contiene espacios) se convierte en true.
Cadenas vacías se convierten en false.
Números:
0, -0, NaN se convierten en false.
Los demás números se convierten en true.
otros valores
Los valores null y undefined se convierten en false.
Objetos y arrays (incluso vacíos) se convierten en true.
